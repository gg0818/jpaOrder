# HELLO FOOD
## 개발동기
* 최근 트렌트인 JPA를 활용한 웹애플리케이션 개발을 해보고자 함 <br>
* 회원가입, 로그인, 주문 서비스를 구현하고자 함 <br>
* 카카오 우편번호 API 사용

## 개발환경
> Frontend : Thymleaf, HTML, CSS, JQuery <br>
> Backend : JAVA, JPA, SpringBoot, MariaDB <br>
> Version Control : Git, Github
