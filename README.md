# HELLO FOOD
## 개발동기
* 최근 트렌트인 JPA를 활용한 웹애플리케이션 개발을 해보고자 함 <br>
* 회원가입, 로그인, 주문 서비스를 구현하고자 함 <br>
* 카카오 우편번호 API 사용

## 개발환경
> Frontend : Thymleaf, HTML, CSS, JQuery <br>
> Backend : JAVA, JPA, SpringBoot, MariaDB <br>
> Version Control : Git, Github

## 느낀점
인프런 김영한 강사님의 "실전! 스프링 부트와 JPA 활용1 - 웹 애플리케이션 개발" 강의를 듣고
무작정 따라서 만들어본거라 아직 부족한 점도 많고 개선되어야 할 점도 많다고 느꼈습니다. <br>
JPA를 공부하면서 테이블 중심으로 생각하는 것이 아니라 객체를 중심으로 생각하는 사고와 그 객체들의 연관관계에 중점을 둔 
객체지향설계에 관해 많이 생각하고 배울 수 있었습니다. ORM 객체 관계 맵핑의 제공으로 단축된 쿼리 작업과, 
간결해진 코드로 비즈니스 로직에 더 집중할 수 있어 매우 편리하다고 느꼈습니다. <br>
단순한 웹 애플리케이션을 작업했는데 더 깊게 공부하여 API 작업과 리팩토링 작업을 해보며 성능 최적화를 해볼 예정입니다.
